# Troubleshooting

This page contains some common issues that you may encounter when trying to play on Xenyria.

If you have an issue that is not listed here, please contact us on [Discord](https://discord.gg/xenyria).

## My game crashes every time I die in PaintSquad

This issue is usually caused by Feather Client. As of right now, there is no fix for this issue. We recommend that you do not use any big client that heavily modify the game. Read our page on [Modifications](/general/modifications) for more information.

## Everything is in the wrong language

Simply run `/language` and select your language. Xenyria currently supports English and German.
